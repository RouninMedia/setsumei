# Define JSON Signature

## Template of Define JSON Signature:

```
/**
 * [ Definition Description ]
 * @typedef {[ Data Type ]} [ Data Description ]
 */
```

## JavaScript Example of Define JSON Signature:

```
/**
 * Define JSON
 * @typedef {string} JSON
 */
```

## TypeScript Example of Define JSON Signature:

```
/**
 * Define JSON
 * @typedef {string} JSON
 */
```
